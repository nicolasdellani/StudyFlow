# 📚 StudyFlow

StudyFlow é uma aplicação web desenvolvida para ajudar estudantes a organizarem seus estudos, tarefas e matérias de forma simples, moderna e intuitiva.

O projeto foi criado utilizando HTML, CSS e JavaScript puro, com foco em experiência do usuário, produtividade e organização acadêmica.

---

## ✨ Funcionalidades

* ✅ Sistema de cadastro e login
* ✅ Dashboard interativa
* ✅ Gerenciamento de matérias
* ✅ Criação e organização de tarefas
* ✅ Prioridades para atividades
* ✅ Marcação de tarefas concluídas
* ✅ Persistência de dados com LocalStorage
* ✅ Interface responsiva e moderna

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage

---

## 🎯 Objetivo do Projeto

O objetivo do StudyFlow é oferecer uma plataforma simples para auxiliar estudantes na organização da rotina acadêmica, permitindo gerenciar tarefas e acompanhar estudos de maneira prática.

Além disso, o projeto também foi desenvolvido como forma de aprimorar conhecimentos em desenvolvimento front-end e lógica de programação.
